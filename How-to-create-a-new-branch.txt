main (Master) - ветка, на  которой лежит основная рабочая версия продукта
git branch  - показать существующие ветки

git branch apple_pay  - создать новую ветку apple_pay
git checkout main - изменить ветку на main, переключиться на др. ветку

git merge apple_pay   - слияния одной или нескольких веток в текущую
